
# Bot Galicia N4 Web

## Cómo usar

1. Instalar dependencias:
   pip install selenium flask

2. Descargar chromedriver.exe (versión Chrome 136) y dejarlo junto al proyecto.

3. Ejecutar:
   python app.py

4. Abrir en el navegador:
   http://localhost:5000

Este bot usa headless Chrome, registra las URLs ya procesadas en calificadas.txt
